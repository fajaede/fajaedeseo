Geavanceerde SEO plugin met AI-ondersteuning, XML sitemap, dynamisch robots.txt beheer, meta tags, OpenGraph, social handles en privacy-tools voor WordPress.

== Description ==

FajaedeSEO AI is een krachtige en gebruiksvriendelijke SEO-plugin voor WordPress met onder andere:
* Automatische XML sitemap op eigen URL (/fajaede_sitemap/)
* Volledig instelbare en dynamische robots.txt
* Focus keywords, contentanalyse, meta title & description per post/page
* Dynamische OG/Twitter tags met social media handles
* Instellingen-dashboard voor eenvoudig beheer en snelle aanpassingen
* Lichtgewicht SEO sidebar voor de Gutenberg editor (posts en pagina's)
* Live SEO meta title, meta description en focus keyword velden
* AI-gestuurde titel- en beschrijvingssuggesties (single-click)
* Real-time SEO scoring met bruikbare feedback en kleurindicaties
* Schrijftips (keyworddichtheid, passieve zinnen, zinslengte, leesbaarheid)
* Contentanalyse (koppen, interne links, afbeeldingen met alt-tekst)
* Keyword suggestie chips gebaseerd op content
* Automatische output van meta tags: standaard, Open Graph, Twitter Cards en meer
* Instelbare fallback afbeelding voor social sharing (geen lege kaarten meer)
* Schema.org Article structured data in JSON-LD voor rijke zoekresultaten
* Gebruik van native block editor UI en volledig JavaScript (React) voor optimale snelheid
* Werkt voor alle publieke posts en pagina’s
* AI functies zoals privacy policy generator, banner optimalisatie en cookiescan (optioneel)

De plugin is lichtgewicht, uitbreidbaar en overschrijft niet de standaard WordPress sitemaps. Privacy compliance en internationale ondersteuning zijn belangrijke speerpunten.

== Installation ==

1. Upload de **fajaedeseo** map naar `/wp-content/plugins/` of installeer via het WordPress dashboard.
2. Activeer de plugin via het menu ‘Plugins’ in WordPress.
3. Beheer site-instellingen, robots.txt en social handles via **Instellingen > Fajaede SEO**.
4. Flush permalinks na activatie (Instellingen > Permalinks > Wijzigingen opslaan).

== Frequently Asked Questions ==

= Waarom krijg ik een 404 op /fajaede_sitemap/? =
Flush permalinks via Instellingen > Permalinks > Wijzigingen opslaan zodat de rewrite rules geactiveerd worden.

= Kan ik eigen social media handles invoeren? =
Ja, je kunt via het instellingen dashboard eigen social media handles toevoegen die automatisch in de OG en Twitter meta tags komen.

= Waar vind ik de sitemap? =
De XML sitemap is beschikbaar op `/fajaede_sitemap/` (bijv. https://jouwdomein.nl/fajaede_sitemap/) en wordt klikbaar getoond in het instellingen dashboard.

= Kan ik de robots.txt aanpassen? =
Ja, de robots.txt wordt dynamisch gegenereerd en is volledig aanpasbaar via de instellingenpagina.

= Heeft de plugin AI functionaliteit? =
Ja, de plugin bevat AI-ondersteuning voor features zoals privacy policy generator, banner-optimalisatie en cookiescan. Deze functies kunnen worden geactiveerd met een API-key.
